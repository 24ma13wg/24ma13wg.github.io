Skills matrix

July, 2018

The above prototype is based on a set of assumed user stories, outlined here. Data capture would be handled by a separate application.

Hover over the data points to see who has the skill. The key indicates how developed/developing their skill level is. They may also be filtered. The data points are generated from a simple, but hopefully versatile, data object:

{
  "name": "Roxy Bloggs",
  "category": "Back end",
  "skill": "Drupal",
  "project": true,
  "student": false,
  "teacher": false
}

It's a work in progress.
